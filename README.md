# GDAI Unity Plugin

Official Unity plugin for GameDevs.AI.

## Installation

- Import the release `.unitypackage` into your Unity project.
- Follow the full setup guide in `SETUP_README.md`.

## Prerequisites

Install these packages before importing:

- `com.unity.nuget.newtonsoft-json`
- `com.unity.ugui` (TMPro APIs are provided by UGUI in Unity 6.4)

## Version

Current public release target: `v0.1.0-alpha.7.1`.
